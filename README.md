# ShopSphere — Full-Stack E-commerce Platform

ShopSphere is a modern full-stack e-commerce web application built with React, Laravel, MySQL, Laravel Sanctum, and Tailwind CSS.

The project includes a complete customer shopping experience and a powerful admin dashboard for managing products, categories, users, orders, promotional offers, and dynamic homepage content.

## Features

### Customer Features

- Register and login
- Browse products
- View product details
- View similar products
- Add products to cart
- Floating cart drawer
- Update cart quantities
- Remove products from cart
- Checkout with shipping details
- View personal order history
- Track order status

### Admin Features

- Admin dashboard
- Manage products
- Create, edit, and delete products
- Upload product images
- Manage product categories
- View customer orders
- Update order status
- Manage users
- Make users admin or remove admin access
- Create homepage promotional offers
- Upload and manage dynamic hero images

## Project Highlights

- Full-stack architecture with separate frontend and backend
- Authentication using Laravel Sanctum
- Protected customer routes
- Protected admin routes
- Admin role management
- Product image upload
- Dynamic homepage hero images managed from the admin panel
- Promotional offers managed from the admin panel
- Floating cart drawer with live cart count
- Checkout and order tracking flow
- Similar products section on product details page
- Modern responsive UI with Tailwind CSS
